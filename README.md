# Angular 17 Admin Dashboard

## Overview
A modern Angular 17 admin dashboard demonstrating real-world frontend architecture.

## Tech Stack
- Angular 17 (Standalone Components)
- Bootstrap 5
- SCSS
- RxJS
- REST API Integration

## Features
- Lazy loaded routes
- Reusable components
- API integration using services
- Reactive forms with validation
- Performance optimizations
- Responsive UI
- Animations (AOS)

## Project Structure
- core: services, guards, interceptors
- shared: reusable UI components
- features: auth, dashboard, users

## How to Run
npm install  
ng serve
